# 🎵 Music Recommendation System

## 📌 Overview
This project is a Music Recommendation System built using Python in Google Colab. It recommends songs based on similarity between music features using machine learning techniques.

---

## 🚀 Features
- Recommend similar songs based on input song
- Uses similarity matrix for fast recommendations
- Built and tested in Google Colab
- Simple and efficient ML-based system

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Pickle

---

## 📂 Project Files
- notebook.ipynb → Main Colab notebook
- musicrec.pkl → Processed dataset
- similarities.pkl → Similarity matrix
- poster/ → Song images folder

---

## ⚙️ How It Works
1. Load dataset in Colab
2. Compute similarity between songs
3. User selects a song
4. System finds similar songs
5. Displays recommendations
